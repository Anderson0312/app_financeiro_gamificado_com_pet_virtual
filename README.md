🐾💰 App Financeiro Gamificado com Pet Virtual

Aplicação web de controle financeiro pessoal com abordagem gamificada, onde o usuário gerencia suas finanças enquanto cuida de um pet virtual, tornando o processo mais engajador e divertido.

✨ Funcionalidades

💸 Controle de receitas e despesas

📊 Visualização de saldo financeiro

🎮 Sistema de gamificação

🐶 Pet virtual que evolui conforme suas finanças

🏆 Recompensas por boas práticas financeiras

📅 Histórico de transações

📈 Incentivo à organização financeira

Aplicações financeiras ajudam usuários a registrar gastos e manter histórico detalhado das transações, melhorando o controle do dinheiro

🧠 Conceito do Projeto

O objetivo do projeto é transformar a forma como as pessoas lidam com dinheiro, utilizando gamificação + emoção (pet virtual) para:

Incentivar hábitos financeiros saudáveis

Tornar o controle financeiro menos “chato”

Criar consistência no uso do app

🎮 Como funciona a gamificação

✔️ Registrar receitas → recompensa o usuário

✔️ Controlar gastos → melhora o status do pet

✔️ Manter consistência → evolução do pet

❌ Falta de controle → impacto negativo no pet

💡 Isso cria um vínculo emocional que incentiva o uso contínuo.
